## 1. Program Knight’s Tour (Open & Closed Tour)

Deskripsi Singkat:
Program menyelesaikan Knight’s Tour pada papan catur 8x8. Kuda berusaha mengunjungi semua 64 kotak tanpa mengulang. Program dapat menghasilkan open tour dan mengecek apakah hasilnya termasuk closed tour.

Cara Menggunakan:
1. Jalankan program.
2. Masukkan koordinat awal kuda dalam format: x y
   Contoh: 0 0
3. Program akan menampilkan:
   - Langkah perjalanan kuda
   - Papan 8x8 berisi urutan kunjungan
   - Posisi terakhir
   - Informasi apakah closed tour atau bukan

Input:
Koordinat awal kuda (x y) dengan nilai 0–7.

Output:
- Daftar langkah
- Papan rute 8x8
- Posisi akhir
- Status closed tour

## 2. Program Largest Monotonically Increasing Subsequence (LIS)

Deskripsi Singkat:
Program menghitung panjang LIS dari daftar angka yang dimasukkan pengguna. Implementasi menggunakan metode binary search (patience sorting).

Cara Menggunakan:
1. Jalankan program.
2. Masukkan deretan angka dalam satu baris, dipisahkan spasi.
   Contoh: 3 10 2 1 20

Input:
Satu baris berisi daftar bilangan bulat.

Output:
Satu angka yang menunjukkan panjang LIS.
